Permissions for the Cat modpack
-------------------------------------------------
1.7.10- Forge

============================
** http://tinyurl.com/cat-permissions **
============================

All mods within this GitHub should be green that list.

Green means permission.
Red means no permission.
Yellow means awaiting permission.

If you find errors in this, please notify CQI immedietly.
If you are a mod author and object to this Github please notify CQI as well.
